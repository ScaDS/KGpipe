# View Package

The view package is a simple streamlit app
to view and visualize the core components of the
framework during development

```
uv run streamlit run src/kgpipe_view/kgpipe_view.py
```

It is different from the KGpipe-Explorer as it only focuses 
on viewing the internal structure and tabular versions of 
the KGpipe Core classes for a connected PipeKG.
